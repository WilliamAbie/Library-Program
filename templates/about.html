Bagaimana cara menggunakan tool ini?

Tool ini merupakan sebuah rancangan sederhana dari sistem pencarian pustaka. Karena tool ini masih dalam proses pengembangan, maka UI dan Programnya mungkin masih belum sempurna dan terdapat bug.

Untuk menggunakan tool ini, silahkan ikuti langkah berikut :
1. Bukalah menu Daftar diatas
2. Pilih sub-menu yang anda inginkan
3. Setelah membuka salah satu dari sub-menu tersebut, maka anda akan disuguhkan data-data dari buku yang tersedia
4. Jika anda ingin menambahkan data, klik "Tambah List"
5. Jika anda ingin mengubah data, klik button "Edit"
6. Jika anda ingin menghapus data, klik button "Hapus"
